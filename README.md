# Background
The application is built using Node.js, with express.js framework and Embedded JavaScript (EJS) for templating. Static files are built to the ./dist folder using Gulp.

## Requirement
```javascript
"node": "^8.1"
```

## Install
```
$ npm install
```

## Usage
```
$ npm run prod
```

## Gulp 
English page
```
$ gulp ejs-english
```
Hindi page
```
$ gulp ejs-hindi 
```
